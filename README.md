# 🎓 AI Student Success Portal

## 📌 Project Overview

The AI Student Success Portal is a comprehensive web-based application developed using Python and Streamlit to support students in monitoring academic performance, wellbeing, personal goals, and overall progress. The platform provides a secure and interactive environment where students can track their activities, analyze performance trends, and receive insights that contribute to academic success.

This project demonstrates the integration of web development, data visualization, database management, and user authentication within a single intelligent student management platform.

---

## 🚀 Key Features

* 🔐 Secure User Registration and Login System
* 🏠 Interactive Welcome Interface
* 📊 Student Performance Dashboard
* 📚 Academic Monitoring Module
* 🧘 Wellbeing and Stress Tracking
* 🎯 Goal Setting and Management
* 🎭 Co-Curricular Activity Tracking
* 📈 Progress Visualization with Interactive Charts
* 💬 Feedback Management System
* 🚪 Secure Logout Functionality
* 💾 SQLite Database Integration
* 🔒 Password Encryption using SHA-256 Hashing
